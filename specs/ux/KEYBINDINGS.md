# Key Bindings

> Default key bindings for ZIGPRESENTERM.

**Status:** 📝 Draft  
**Owner:** @team  
**Date:** 2026-03-01

---

## Navigation

| Action | Default Keys |
|--------|--------------|
| Next slide | `l`, `j`, `→`, `↓`, `Space`, `Enter`, `PageDown` |
| Previous slide | `h`, `k`, `←`, `↑`, `Backspace`, `PageUp` |
| First slide | `gg` |
| Last slide | `G` |
| Go to slide | `<number>G` |
| Next (fast) | `n` |
| Previous (fast) | `p` |

## Actions

| Action | Default Key |
|--------|-------------|
| Execute code | `Ctrl+e` |
| Reload file | `r` |
| Toggle slide index | `Ctrl+p` |
| Toggle key bindings help | `?` |
| Toggle visual grid | `T` |
| Exit | `q`, `Ctrl+c` |
| Suspend | `Ctrl+z` |
| Close modal | `Esc` |

## Modals

When a modal is open, most navigation is disabled:

| Action | Key |
|--------|-----|
| Close modal | `Esc`, `q` |
| Navigate in index | `↑`, `↓`, `Enter` |
